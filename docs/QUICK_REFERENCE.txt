┌─ DATABASE STRUCTURE ──────────────────────────────────────────────────────┐
│                                                                              │
│  Database: "kiddotubes"                                                     │
│                                                                              │
│  Collections:                                                               │
│  • users              User profiles + watch history + favorites            │
│  • filterlogs         Content filtering audit trail                        │
│  • videos             Video metadata cache (optional)                      │
│  • sessions           Active user sessions (optional)                      │
│                                                                              │
│  Watch History: Keeps last 100 entries (auto-cleanup)                     │
│  Fallback: Uses localStorage if MongoDB unavailable                       │
│  Backup: Automatic MongoDB Atlas backups                                  │

┌─ TROUBLESHOOTING ─────────────────────────────────────────────────────────┐
│                                                                              │
│  Problem: "Cannot connect to MongoDB"                                      │
│  Solution: Check internet, verify IP whitelisted, confirm credentials      │
│                                                                              │
│  Problem: "MongoDB backend unavailable"                                    │
│  Solution: Run `npm run server:dev`, app falls back to localStorage        │
│                                                                              │
│  Problem: "Port 5000 already in use"                                       │
│  Solution: Kill process or set PORT=5001 npm run server:dev               │
│                                                                              │
│  Problem: "Module not found: mongoose"                                     │
│  Solution: Run `npm install`                                               │
